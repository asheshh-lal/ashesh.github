Individual Player Analysis

  • Tech Stack: Python, HTML, CSS, Bootstrap, Django, Pandas, Numpy, statsbombpy, mplsoccer, matplotlib, Docker
  • Github URL: Project Link

The project involved creating a web app using the Django framework, where users could upload football event data from StatsBomb. The app analyzes the data for two teams to generate pass maps, shot maps, heat maps, and expected goals.

Additionally, users can select individual players for detailed analysis, providing insights into performance metrics such as passes made, types of passes, shots, shot distances, and nearby players during key events.

The dataset used in the project contained data for 4,000 event points, covering various aspects of match play. This extensive data allowed for in-depth analysis of team strategies and player behaviors.

Docker was utilized to containerize the web app, ensuring that the environment was consistent and easily deployable across different systems. This allowed for seamless integration of the necessary dependencies and ensured that the app could be run reliably in any environment.

The web app offers an intuitive interface, enabling users to upload data, select specific matches, and explore different visualizations that help in understanding the dynamics of a football game.

By combining advanced data analytics with interactive visualizations, the project provides valuable insights to coaches, analysts, and fans, helping them break down the game and make data-driven decisions.
